Quality Hierarchy — Best → Worst
#1
4K Remux H.265 · HDR10+ · TrueHD 7.1.4 Atmos — Perfect fidelity, largest files. Use Shield TV Pro via soundbar HDMI IN.
#2
4K High-Quality Encode H.265 · HDR10+ · TrueHD Atmos · <20 GB — 95% of remux quality, reasonable size.
#3
4K WEB-DL H.264/H.265 · HDR10+ · EAC-3 Atmos · 8–20 GB — Best choice for Google TV chain.
#4
1080p Remux H.265 · TrueHD · 10–25 GB — Excellent quality. Use Shield for lossless audio.
#5
1080p High-Quality Encode H.265 · TrueHD or DD+ · 3–8 GB — Very good, practical size.
#6
Screener / TS — Watermarks, audio sync issues. Avoid unless desperate.
#7
CAM — Unwatchable. Never recommended.
TrueHD Atmos remuxes are only as good as their audio chain. Use the Shield TV Pro (soundbar HDMI IN) to actually benefit. On the Google TV chain, DD+ Atmos is the ceiling regardless of file quality.
Understanding Stream Information View

Every stream listing in Nuvio/Stremio shows a compact block of icons and labels. Here's what each one means so you can evaluate quality at a glance before tapping play.

Main Line
⚡ / ⏳ Debrid Cache Status
⚡ = Cached (instant playback) · ⏳ = Not cached (may take longer to start)
🧲 / 🌐 / 📺 Stream Source Type
🧲 = Torrent (P2P) · 🌐 = Direct HTTP · 📺 = Live stream
▶️ / 🗄️ / ↗️ / 📊 / ℹ️ / ⛔ Special Stream Type
YouTube · Archive · External · Statistics · Information · Error
UHD ⁴ᴷ / QHD ²ᴷ / FHD / 720P Resolution
UHD 4K > QHD 2K > FHD 1080p > 720p
⌜QUALITY⌟ Source Quality Tag
Remux · WEB · WEB-DL · BluRay · HDTV · etc.
◆◆⬖◇◇ Release Quality Score
Based on Vidhin's Ranked Regexes — sorted by quality & resolution. More filled diamonds = better release.
Technical Details
▶︎ / 🎬 Edition
Director's Cut · Extended · IMAX · Theatrical · etc.
▣ / 📼 Video Encoding
x264 · x265 · HEVC · AV1 · etc. H.265/HEVC is preferred — better quality at smaller file size.
✦ / 🎥 Visual Features
HDR · HDR10+ · Dolby Vision · 10-bit · etc. S90D supports HDR10 and HDR10+ — not Dolby Vision.
♬ / 🎵 Audio Format
TrueHD · DTS-HD MA · DD+ Atmos · EAC-3 · AC3 · AAC — see Audio section below.
☊ / 🎧 Audio Channels
2.0 · 5.1 · 7.1 · 7.1.4 — more channels = more immersive if your system supports it.
File & Availability
◧ / 💾 or ⧉ / 📚 File Scope
Single file · Season pack — season packs contain all episodes in one torrent.
Size · Bitrateᴹᵇᵖˢ File Size & Bitrate
Helps estimate quality vs. bandwidth. Higher bitrate = more data = better quality (usually). Remuxes are largest; encodes compress down.
⟳ / 🕒 Upload Age
When the torrent or file was uploaded. Newer releases are often better seeded and more likely to have current encodes.
Provider & Delivery
⛊ / ⛉ or 🔐 / 🔧 [Provider] Debrid Service & Scraper
Which debrid (e.g. RealDebrid) and which addon scraped the result. Proxied vs. unproxied affects routing.
TYPE Stream Type Label
P2P · HTTP · LIVE · USENET · NNTP · YOUTUBE · ARCHIVE · EXTERNAL · STATISTIC · INFO · ERROR
⇋ Seeders 𖧧 / 🌱 Seeder Count
P2P only. Higher seeders = more reliable download / stream start. On debrid with ⚡, seeders matter less since the file is already cached.
Languages
⚐ / 🔊 Available Audio Languages
Flag icons or audio speaker icon shows which dub languages are included in the file.
☰ / 💬 Available Subtitle Languages
Which subtitle tracks are embedded or available.

SeaDex's top pick for this title.
[ALT] Strong Alternative
A reliable backup if the [BEST] release isn't available or fails to load.
Additional
✎ / ✏️ FILENAME Filename
The actual torrent or file name — useful for identifying the exact release group.
ⓘ / ℹ️ MESSAGE Informational Message
Additional context from the addon — e.g. content warnings, availability notes.
👉 Quick Pick Guide With debrid: Prioritize ⚡ cached streams first, then highest resolution, then strongest ◆ score, then reasonable file size.
On P2P (no debrid): Prioritize high seeder count (⇋), then resolution, then ◆ score. Uncached or low-seeder streams may buffer or fail to start.
