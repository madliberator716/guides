Lossless Music
Cheat Sheet

Formats, spectrograms, codecs, and what actually matters for your iPhone + Bose setup.

Core Definitions
✅ Lossless
Preserves 100% of original data
FLAC, ALAC, WAV, AIFF
No data removed
❌ Lossy
Permanently removes data to save space
MP3, AAC (M4A), Opus
Can never be restored to lossless
CriticalConverting lossy → FLAC/WAV does NOT make it lossless. The damage is permanent.

Bitrate vs Sample Rate vs Bit Depth
kbps (bitrate) — Data per second. Only applies to lossy. Higher ≠ lossless.
kHz (sample rate) — Max frequency = sample rate ÷ 2. At 44.1 kHz → max ~22 kHz (human hearing limit).
Bit depth — Dynamic range detail. 16-bit = CD quality. 24-bit = more headroom (not always audible).
Spectrogram — How to Tell Real from Fake
✅ True Lossless
High frequencies fade gradually
Energy continues past 16–18 kHz
No straight horizontal cutoff
Grainy/noisy texture above ~18 kHz
❌ Lossy (or Fake Lossless)
Hard "brick wall" cutoff ~16–18 kHz
Everything above cutoff = empty / dark
Smooth flat line → codec filtering

Wall = Lossy  ·  Fade = Lossless

This is the single rule that reveals the truth of any file

MP3 vs AAC
AAC (M4A) — Better
Newer codec
Smoother highs
Better quality at same bitrate
Optimized for Apple devices
Target: 256 kbps
MP3 — Worse
Older codec
Rougher/grainier highs
Less efficient
If unavoidable: 320 kbps
At the same kbps: AAC always beats MP3
Your Setup — iPhone + Bose (Bluetooth)
📡
Bluetooth is always lossy (AAC codec)
🔇
You will not hear lossless over Bluetooth — it's re-encoded regardless
✅
Best choice for iPhone + Bluetooth: AAC 256 kbps
📦
Keep lossless only if you want a master archive copy
Apple Music Lossless only matters for:Wired headphones · External DAC · Storage/archive quality
Important Truths
⚠️
Loud spectrogram ≠ better quality
⚠️
Dark spectrogram ≠ lossy
⚠️
Different spectrograms ≠ different formats
⚠️
Same song can look different due to mastering
⚠️
Brightness often = EQ + compression, not quality
What to Keep
✅
Streaming / iPhone use → AAC 256 kbps
✅
Archival / mastering / collection → FLAC / ALAC
🗑️
Delete "FLACs" with a hard cutoff → fake lossless
Final TakeawayLossless is about data integrity — not how loud, bright, or expensive it looks. Spectrograms reveal the truth.
